Backup jetzt starten! Startet ein Backup aller konfigurierter Miniserver.

Downloads: Zum Herunterladen vorhandenen Backup-Archive in einem neuen Fenster.

Logdatei speichern: Zeigt die Logdatei der letzten Backup-Durchläufe in einem neuen Fenster an.

Automatische Backups: Schaltet automatische, zeitgesteuerte Backups ein.

Zyklus: Legt fest, wie häufig das Backup ausgeführt werden soll.

Anzahl Backups im Archiv speichern: Anzahl an älteren Backups, die vorgehalten werden sollen. Bitte beachten, dass genügend Speicherkapazität auf dem Laufwerk zur Verfügung stehen muss.

Protokollierungs-Stufe: Auswahl, wie detailliert das Protokoll (Logdatei) gespeichert werden soll.

Logdatei live anzeigen: Zeigt die aktuelle Logdatei an ab dem Zeitpunkt des Anklickens an.

Logdatei aktualisieren: Aktualisiert die aktuelle Logdatei laufend.
Durch wiederholtes Klicken oder nach 15 Minuten wird die Aktualisierung pausiert.

Automatisch zum Ende springen: Das Logdatei rollt automatisch ans untere Ende, wenn Einträge hinzukommen.

Debug-Sitzung neu starten: Startet eine neue Debug-Sitzung. Das bedeutet, dass Position in der Logdatei auf den Zeitpunkt des Anklickens gesetzt wird. Wurde die Anzeige vorher pausiert, fehlt das Stück, während die Anzeige pausiert war. (Im Normalfall wird das die Logdatei ab dem Zeitpunkt des Pausierens am Ende der Pause fortgesetzt.)

Die Änderung der Protokollierungs-Stufe wirkt sich nicht auf bereits laufende Datensicherungen aus! Sie wird für den nächsten Durchlauf verwendet.
