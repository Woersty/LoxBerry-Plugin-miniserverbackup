Start Backup now! Starting a Backup of all configured Miniservers.

Downloads: To dowlload existing Backup-Archives in a new Window.

Save Logfile: Shows the Lofile of the last Backup-Runs in a new Window.

Scheduled Backups: Enables automatic, scheduled Backups.

Interval: Defines the Interval for the scheduled Backups.

Number of backups to be stored in the archive: Number of older Backups to be kept. Please ensure, the the storage device has enough capacity.

Logging-Level: Select, how detailed the Logfile should be.

Logdatei Liveview: Shows the current Logfile starting at the time of your click.

Refresh Logfile: Refreshs the current Logfile continously.
The refresh is paused by clicking again or automatically after 15 minutes.

Autoscroll Logfile: The Logfiles is automatically scrolled to the bottom, if a new line is added.

Restart Debug-Session: Starts a new Debug-Session. That means, the position in the Logfile is set to the time of your click. If the Logview was paused before, the lines during the pause will be supressed. (In the normal case the logfile continues at the pause timestamp.)

Changing the Logging-Level has no effect on already running Backups! It will be taken into account for the next run.
